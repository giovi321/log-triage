Add or remove users

  1. Open the config editor and locate webui.admin_users.
  2. Each entry needs username and a bcrypt password_hash.
  3. Save and reload; the Web UI applies the updated list immediately.

Generate a bcrypt hash

Use the password changer on the Password page to produce a new bcrypt hash for your user, or run python -m passlib.hash.bcrypt -e "yourpass" on the server.

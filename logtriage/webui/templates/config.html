Editing tips

  • modules: map log files to a pipeline. Each module can be enabled/disabled and points to a path on disk.
  • pipelines: combine a classifier and grouping/alerting logic. Use the regex lab to refine classifier regexes.
  • webui: controls access. Keep secret_key long and random, and manage admin_users here.
  • alerts: configure sinks such as Slack, email, or stdout. Make sure credentials are set via environment variables when supported.
  • Use the config editor to apply changes safely with validation and automatic backups.
